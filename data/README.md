This project uses a public car sales dataset available on Kaggle.

Source: https://www.kaggle.com/datasets/missionjee/car-sales-report

Description: The car sales dataset contains vehicle sales records from multiple dealerships, including information such as dealership, vehicle attributes, and sales values.

The dataset is publicly available and maintained by the original author on Kaggle.

The raw data is included as part of the Excel workbook in the /excel folder and was used to create pivot tables, visualizations, and an interactive dashboard.
